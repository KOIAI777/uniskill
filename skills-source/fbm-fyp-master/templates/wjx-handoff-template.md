# WJX HANDOFF PACKAGE Template

当用户完成问卷设计后，输出以下 handoff 包，供问卷星配置或 `wjx-auto-fill` 工作流使用。

## 1. Study Goal

- 研究题目
- 核心研究问题
- 实验 / 问卷类型

## 2. Conditions

| Condition ID | Scenario | Style | Notes |
|---|---|---|---|

## 3. Block Structure

| Block | Purpose | Question Type | Estimated Items |
|---|---|---|---|

## 4. Variable Mapping

| Variable | Item IDs | Reverse? | Scale |
|---|---|---|---|

## 5. Manipulation Checks

- `M1:`
- `M2:`

并说明每个条件的正确答案。

## 6. Cleaning Rules

- 缺失值剔除标准
- manipulation check 失败标准
- 异常值处理规则

## 7. Expected Quotas

- 总样本数
- 每条件目标样本
- 保底样本

## 8. Autofill Hints

输出给自动填写流程的要点：

- 题型概览
- rate matrix 是否存在
- reverse item 列表
- 单选 / 多选 / 文本题位置
- 哪些题必须精准命中条件答案
