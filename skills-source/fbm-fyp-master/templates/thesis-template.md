# Chinese Thesis Template

按以下顺序输出中文论文草稿：

## Cover Page Information

- 题目
- 姓名 / 学号 / 专业 / group / supervisor / date

## Acknowledgement

- 可先给占位版

## Abstract

- 约 200-250 词
- 包含：研究目的、方法、样本、主要结果、意义

## Keywords

- 3-5 个关键词

## 1. Introduction

- 背景
- 问题
- 研究意义
- 研究目标

## 2. Literature Review

- 分主题综述
- 研究缺口

## 3. Research Design / Methodology

- model
- hypotheses
- sample
- instrument
- procedure
- analysis methods

## 4. Data Analysis And Results

- cleaning
- reliability
- descriptive statistics
- hypothesis testing
- APA 7 三线表

## 5. Discussion

- 解释结果
- 与文献比较
- 管理启示 / 理论意义

## 6. Conclusion

- 总结
- limitations
- future directions

## References

- APA 7

## Appendices

- questionnaire
- extra tables
- technical notes

## Table Reminder

正文里所有正式统计表默认采用 APA 7 三线表风格：

- `Table X`
- 表题
- 三线表结构
- `Note.`
