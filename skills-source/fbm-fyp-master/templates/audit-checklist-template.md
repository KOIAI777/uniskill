# Final Audit Checklist Template

## High Priority

- 占位符未删除
- 样本量 / 数字前后不一致
- 表图编号错误
- 文中引用与参考文献不匹配
- hypothesis / variables / questionnaire / analysis 不一致
- Declaration Form 缺失或不是官方版式
- TOC 未更新
- 弱来源承担核心理论依据

## Medium Priority

- 标题层级不一致
- abstract 与正文不完全对齐
- discussion 没有回到 hypotheses
- limitations 太弱
- `Note.` 样式不对
- 新增 references 未继承 hanging indent
- References 排序错误

## Low Priority

- 行文重复
- 个别术语不统一
- 图表标题风格不一致
- figure / table note 可读性一般

## Fix Order

1. 先修事实错误
2. 再修变量与模型一致性
3. 再修 Declaration / TOC / References
4. 再修 APA
5. 最后修语言表达
