# Data Analysis Report Template

## 1. Data Cleaning Report

- 原始样本量
- 清洗后样本量
- 剔除规则
- 各条件保留率

## 2. Variable Construction

| Variable | Formula | Items | Notes |
|---|---|---|---|

## 3. Descriptive Statistics

| Variable | Mean | SD | Min | Max |
|---|---|---|---|---|

## 4. Reliability

| Scale | Cronbach's alpha | Judgment |
|---|---|---|

## 5. Correlation Analysis

- 相关矩阵
- 高相关提醒
- 简短解释

## 6. Main Hypothesis Tests

每个假设固定输出：

- 方法
- 模型
- 核心统计量
- p 值
- effect size
- 是否支持
- 一句中文结论

## 6.1 APA 7 Three-Line Tables

把关键统计结果整理成论文可用的 APA 7 三线表，优先输出：

- descriptive statistics table
- reliability table
- correlation matrix
- hypothesis testing summary table
- ANCOVA / regression / mediation summary table

每张表固定包含：

- `Table X`
- 表题
- 字段布局
- `Note.`

## 7. Thesis-Ready Results Paragraph

把关键结果整理成可写入论文的中文结果段落，但不得新增不存在的数据。

## 8. Managerial / Academic Takeaway

- 一句话总结发现
- 对实践 / 理论的意义
