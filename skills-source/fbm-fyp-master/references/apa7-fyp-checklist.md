# APA 7 Checklist For FBM FYP

本清单用于 Proposal 轻量 APA 检查与 Final Paper 全量 APA 检查。

## A. In-Text Citations

检查：

- narrative citation 是否正确  
  `Author (Year)`
- parenthetical citation 是否正确  
  `(Author, Year)`
- 三位及以上作者是否使用 `et al.`
- 直接引语是否带 page number
- 多来源并列是否按 APA 排序和分号分隔
- 同作者同年是否区分 `2024a / 2024b`

## B. Reference List

检查：

- 作者名格式
- 年份括号
- 题名大小写
- 期刊名与卷号斜体
- issue 放在括号内
- DOI 用 `https://doi.org/...`
- URL 是否为正式链接
- 文末按作者字母顺序排序
- 是否使用 hanging indent
- 后补条目是否也继承 hanging indent
- 中文参考文献是否排在英文条目之后

## C. Cross-Check

逐项核对：

- 文中出现但参考文献缺失
- 文末列出但正文未引用
- 作者拼写不一致
- 年份不一致
- 强来源与弱来源是否被混用为同等理论依据

## D. Abstract And Keywords

检查：

- Abstract 是否单段
- 是否约 200-250 词
- Keywords 是否紧跟 abstract
- `Keywords:` 标签格式是否统一

## E. Heading Levels

检查：

- Heading 层级是否一致
- 同层标题格式是否一致
- 是否存在跳级

## F. Tables And Figures

检查：

- Table / Figure 编号是否连续
- 标题是否清晰
- note 是否补充必要信息
- 正文是否引用每个表图

### APA 7 三线表专项

表格默认按 APA 7 的简洁三线表思路检查：

- 顶线 / 表头下线 / 底线
- 不要竖线
- 不要满网格边框
- 表号 `Table X` 是否规范
- 表题是否位于表号下方
- 小数位是否统一
- 显著性标记是否统一
- `Note.` 是否解释缩写、样本量、显著性
- 表格内容是否与正文一致
- 复杂统计表是否需要一条额外横线分块

## G. Appendices

检查：

- Appendix A / B / C 是否连续
- 标题是否明确
- 正文是否提及附录

## H. .docx Formatting

检查：

- 字体
- 字号
- 双倍行距
- 首行缩进
- 2.5 cm 页边距
- reference entries hanging indent
- TOC 是否与正文一致
- `Declaration Form` 是否已放在 Acknowledgement 前

## I. Notes

检查：

- 只有 `Note.` 标签斜体
- `p` 等统计符号是否单独斜体
- `Note.` 是否不首行缩进
- Figure note / Table note 是否风格统一

## Output Format

APA 检查输出固定分成两部分：

### 1. APA 问题清单

- 问题位置
- 问题类型
- 为什么不符合 APA 7

### 2. 修正后的标准写法示例

- 给出标准文内引用
- 给出标准 reference entry
- 必要时给出标题、表图、附录的标准样式
- 如果是表格问题，给出修正后的 APA 7 三线表字段布局
