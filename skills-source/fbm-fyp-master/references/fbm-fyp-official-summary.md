# FBM FYP Official Summary

本文件整理自 FBM FYP Operational Manual、FYP FAQ 与 monthly report 模板，供 Skill 在回答时引用。

## 1. 课程与时间线

- FYP 为 `BUS4073 + BUS4083` 年度课程。
- 第一学期重点：确定题目、proposal、workshops、monthly reports。
- 第二学期重点：数据收集、分析、draft、final report、oral presentation。

常见关键时间点：

- Sept / Oct / Nov：monthly reports + proposal
- Mar / Apr：monthly reports + first draft + final report
- Apr 30：final report 截止
- May：oral presentation

## 2. 课程核心目标

FYP 应体现：

- conduct a small-scale business research project
- use appropriate methodology
- write with proper references and citations
- show analytical and intellectual capability

## 3. Final Report 基本格式

- 正文通常不超过 30 页
- 不含 cover page、acknowledgement、table of contents、abstract、references、appendices
- A4
- 双倍行距
- 12 号字体
- 四边 2.5 cm
- paginated
- Abstract 约 200-250 词
- References follow APA
- Declaration Form 放在 Acknowledgement 前
- TOC 推荐自动生成
- Abstract 单独一页

## 4. Final Report 典型结构

- Title page
- Declaration form
- Acknowledgement
- Table of contents
- Abstract + keywords
- Introduction
- Main text
- Conclusion
- References
- Appendices

## 4.1 Declaration And Release Forms

手册明确提到：

- `Appendix 2: BBA Project Declaration Form`
- `Appendix 3: BBA Project Release Form`

系统应默认提醒：

- 最终稿需要包含 Declaration Form
- Declaration Form 不能只理解为“一段 declaration 文本”
- 若用户处于最终提交阶段，最好接近官方附录样式

## 5. Turnitin 与 AI 风险

- proposal / report 会经过 Turnitin
- similarity index 过高可能被打回
- AI-assisted writing tools generally prohibited
- final submission 也可能经过 AI detection

因此，Skill 输出应始终视为：

- brainstorming
- drafting support
- structure aid
- APA / audit support

而不是替代学生最终提交责任。

## 5.1 References And Chinese Sources

手册明确要求：

- `References: Follow the APA format for all reference entries.`
- 中文参考文献不宜过多
- 中文参考文献需翻译成英文并保留中文题名
- 中文参考文献应排在英文条目之后

## 5.2 Manual-Implied Checks

虽然手册没有逐字列出每一个 APA 细节，但由于其明确要求遵循 APA 7，系统应默认检查：

- alphabetic ordering of English references
- hanging indent
- in-text citation / reference-list matching
- table / figure note conventions

## 6. Monthly Report 要点

monthly report 需要体现：

- 本月进度
- 与导师 meeting 内容
- 已完成研究活动
- 遇到的问题
- 导师建议如何落实
- research activity journal，通常要求较充分展开

## 7. Oral Presentation 要点

- 常见总时长 35 分钟左右
- presentation 约 20 分钟
- Q&A 约 15 分钟
- 小组成员都要参与
- 逻辑组织、表达、团队分工、时间控制、visual aids 都会影响评分
