# EBIS / Questionnaire Research Pattern

本文件总结问卷实验型 FYP 的高复用流程，尤其适合 EBIS、MKT、FIN 等使用 questionnaire / experiment / scenario vignette 的研究。

## Typical Pipeline

1. 选题与专业相关
2. 文献检索
3. research gap
4. RQ / Hypothesis
5. variables and operationalization
6. questionnaire or scenario design
7. pilot logic / manipulation checks
8. data collection
9. data cleaning
10. reliability and descriptive statistics
11. main model testing
12. thesis writing

## Good Topic Traits

- 与专业课程或行业场景强相关
- 可以通过问卷、实验、二手数据验证
- 有足够近五年的学术文献
- 变量关系清晰
- 可在一个学期内完成

## Common Variable Blocks

- Independent variable
- Dependent variable
- Mediator
- Moderator
- Control variables
- Manipulation checks

## Common Survey Output

- demographic block
- screen / filter questions
- scale items
- reverse items
- manipulation checks
- open-ended optional item

## Common Analysis Output

- cleaning log
- sample profile
- reliability
- descriptive statistics
- correlations
- main hypothesis tests
- effect interpretation
- thesis-ready result wording

## Writing Principle

- 文献先行
- 变量逻辑先锁定
- 问卷题项与变量一一映射
- 结果章节只写已有统计输出
- discussion 回到假设与文献，不新造数据
