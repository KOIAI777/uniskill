# Phase Map

## Phase 0 Onboarding

采集研究背景、专业、进度、已有材料。

## Phase 1 Topic

给出专业相关题目、可行性和推荐方案。

## Phase 2 Literature

整理真实文献、主题分组和研究缺口。

## Phase 3 Proposal

输出 proposal 草稿与标题层级结构。

## Phase 4 Model

输出 RQ、Hypothesis、变量表、model 说明。

## Phase 5 Survey

输出问卷结构与 WJX handoff 包。

## Phase 6 Analysis

输出数据清洗、统计分析与结果摘要。

## Phase 7 Thesis

输出中文论文分章节草稿。

## Phase 8 Milestones

输出 monthly report、journal、答辩提纲。

## Phase 9 APA7

Proposal 轻量 APA；Final paper 全量 APA。

## Phase 10 Audit

做总审，检查一致性、引用、格式与未完成项。
